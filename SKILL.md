# Workplace Resilience Skill

A tactical skill module for identifying, countering, and documenting workplace bullying — with specific playbooks for Slack, Feishu/Lark, and Microsoft Teams.

## Quick Start

1. **Identify the attack type** — cold violence, chat exclusion, manipulation, or eccentric behavior
2. **Pick your counterpunch** — from the platform-specific tables or the 10-move playbook
3. **Document immediately** — screenshot, timestamp, permalink
4. **Escalate with evidence** — never with feelings alone

## Core Counterpunch Moves (Quick Reference)

| Move | Trigger | One-Line Summary |
|------|---------|-----------------|
| Paper Trail Maneuver | Verbal/DM agreement likely to be denied | Post public recap within 30 min |
| Factual Mirror | Hostile or false statement made about you | "Are you saying [exact words]? Is that accurate?" |
| Precision Timestamp | Gaslighting — "That never happened" | Go to receipts immediately; never argue memory |
| Channel Redirect | Moved to DM to remove witnesses | Loop in a third party or post a public summary |
| Redirect to Process | Pressured to act off the record | "Happy to move — please put it through [official channel]" |
| Strategic Silence Break | Systematically ignored in a channel | Direct targeted message → escalate if still ignored |
| CC Cascade | Information withheld from you, shared with peers | Reply to what you do receive; add manager for visibility |
| Boundary with Teeth | Repeated disrespect in chat or meetings | Exact quote + date + professional ask, in writing |
| Volunteer Witness | Anticipate disputed 1:1 meeting | Offer to include a third party before the meeting |
| Defuse and Document | Outburst or attack in group setting | De-escalate publicly; document and formalize immediately after |

## Platform-Specific Attack Patterns

### Slack
- Channel ghosting (coordinated ignore)
- Selective @mention (credit omission)
- Private channel exile
- DM trap (no witnesses)
- Thread burial
- Read-and-ignore power move
- Passive-aggressive reaction bombing

### Feishu / Lark
- Group chat exile
- Approval chain weaponization (deliberate delay)
- Document permission lock (view-only while peers edit)
- OKR comment ambush (last-minute negative reviews)
- Meeting link omission
- Read receipt gaslighting

### Microsoft Teams
- Channel removal without notice
- Message-read denial
- Meeting exclusion with plausible deniability
- Decision-in-chat-you-weren't-in

## Documentation Standard

Every incident log entry must capture:

```
Date/Time:      [platform timestamp]
Platform:       [Slack / Feishu / Teams / email / in-person]
Who:            [names and roles]
What happened:  [exact quotes, not paraphrases]
Witnesses:      [names, or "none"]
My response:    [what you said or did]
Evidence saved: [screenshot path / permalink / email forward]
Impact:         [effect on your work]
```

Store all evidence **outside company systems** (personal cloud storage, personal email).

## Escalation Path

```
Incident occurs
     │
     ▼
Document immediately (screenshot + log entry)
     │
     ▼
Counterpunch (appropriate move from playbook)
     │
     ▼
Pattern continues? → Direct conversation with documented follow-up
     │
     ▼
No resolution? → Manager escalation with dossier
     │
     ▼
Manager unresponsive/complicit? → HR formal complaint
     │
     ▼
HR fails? → Senior leadership / external options (labor board, legal)
```
